# MoCap-Studio
Suivez les étapes pour utiliser MoCap Studio :
# chapitre 0 #1 : Téléchargement de PHP
étape 1 : télécharger le fichier ZIP PHP dans l'onglet "VS17 x64 Thread Safe" du site https://www.php.net/downloads.php?os=windows

étape 2 : dezipper le fichier zip.

étape 3 : trouver et remplacer le fichier "php.ini-development" par "php.ini".
# chapitre 0 #2 : ajouter PHP à l'environnement PATH
étape 1 : taper dans la barre de recherche en bas à gauche "variable environnement"

étape 2 : aller dans "variables d'environnments"

étape 3 : cliquer sur path puis modifier

étape 4 : cliquer sur Nouveau puis mettre le chemin d'accès

étape 5 : cliquer sur ok puis ok puis appliquer ou ok
# chapitre 0 #3 : vérifier qu'il y a PHP
étape 1 : metter dans la barre de recherche cmd ou invite de commande

étape 2 : mettre php -v

vous de veriez avoir un truc comme PHP 8.x.x (cli)
# chapitre 1 #1 : préparation du setup
étape 1 : cliquer sur code puis sur télécharger le fichier ZIP

étape 2 : dézipper le fichier ZIP
# chapitre 1 #2 : préparation du CMD & activation du localhost
étape 1 : mettre la commande suivante : "cd C:\ton\chemin\d'accès\verre\ton\dossier\MoCap Studio

étape 2 : mettre la commande php -S localhost:8080

étape 1 : ouvrir votre navigateur internet
